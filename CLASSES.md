A programação orientada a objetos é uma das abordagens
mais eficientes para escrever software.

Na programação
orientada a objetos, escrevemos classes que representam
entidades e situações do mundo real, e criamos objetos
com base nessas classes. Quando escrevemos uma classe,
definimos o comportamento geral que toda uma
categoria de objetos pode ter.

Quando criamos objetos individuais a partir da classe, cada objeto será
automaticamente equipado com o comportamento geral; então você
poderá dar a cada objeto as características únicas que desejar.



Uma classe descreve um objeto como se fosse um modelo daquele objeto

Ex: Se formos descrever uma pessoa, ela tem nome,alura, peso, rg, cpf 
endereço e tudo isso é definido na classe

## Criando e usando uma classe

Podemos modelar de tudo usando classes. Vamos escrever uma classe simples, 
Pokemon, que representa um pokemons – não um pokemon
em particular, mas qualquer pokemon. O que sabemos sobre a maioria
dos pokemons? Bem, todos eles têm um nome, um tipo, e um level.
Também sabemos que a maioria deles evoluem.
Essas informações nome, tipo e level e o comportamento de evoluir, setão
utilizados no nosso exemplo de classe Pokemon, pois são comum a maioria
dos pokemons
Assim a classe dirá ao python como cria um objeto que represente um Pokemon.
Depois criaremos intancias individuais que representará Pokemons específicos.
